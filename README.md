# Hand-Tracker:

Hello!
This is a programme useful for physics teachers to make students understand the energy category more interactive.
You can record kinetic energy and mechanical work made by your hand in a record time.
To use this you need a webcam python 3-12 installed and windows.(or go to the releases menu and there you will find the .exe file)

# Destination:

This programme is made for Romanian teachers but if you want a tutorial check the ## How to use the programme part in README.md

## Technical information:

It is using mediapipe for the hand and everything is made with python 3.12.

## How to run the programme:
To run the application, follow these steps:
- Download hand_tracker.zip archive from the Releases menu; link: https://github.com/MATEII96/HandTracker/releases.
- Extract the file (right-click the archive -> Extract All).
- Open the extracted folder and run hand_tracker.exe.
Note: The application depends on all the files in the folder to function correctly, so ensure you do not move the .exe file out of the folder.

## How to use the programme:

To exit the programme press Q.
To record energy press R.(to stop recording press R again)
To make sure it works you will need a webcam.
